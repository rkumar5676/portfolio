Zomato Sales Analysis Dashboard
A multi-page Power BI dashboard analyzing restaurant performance, order trends, city-wise revenue, and customer ratings across 100K+ restaurants and 500K+ records.
💡 About this Project: Built in Power BI Desktop using Zomato public dataset. Features 5 dashboard pages: Overview, City Analysis, Restaurant Performance, Rating Analysis, and Trend View. Uses DAX measures, Power Query ETL, and Star Schema data model.
🐈 View Full Project on GitHub 🏠 Visit Portfolio

🎯 Key Business Insights Delivered

📍 City-wise Revenue Breakdown
Identified top 10 revenue-generating cities with drill-through capability
📈 Order Trend Analysis
Year-over-year order volume and revenue growth tracked with DAX time intelligence
⭐ Rating vs Revenue Correlation
Scatter analysis showing rating impact on order volume across restaurant categories
🍕 Top Restaurant Categories
Category-level performance ranking with dynamic filtering and slicer interactivity
